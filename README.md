# Portfolio

A modern, responsive portfolio website showcasing projects and skills as a web developer. Built with React, Vite, and TypeScript for optimal performance and developer experience.

**Live Demo:** [kaushalsahu.tech](https://www.kaushalsahu.tech/)

## Features

- ✨ **Responsive Design** - Works seamlessly on all devices (mobile, tablet, desktop)
- 🎨 **Modern UI/UX** - Beautiful glassmorphic design with smooth animations
- ⚡ **High Performance** - Built with Vite for lightning-fast load times
- 🎭 **Smooth Animations** - Framer Motion and GSAP animations
- 📱 **Mobile Optimized** - Tailwind CSS for responsive layouts

## Tech Stack

- **Frontend Framework:** React 19
- **Build Tool:** Vite 6
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4, Tailwind Animate
- **Animations:** Framer Motion, GSAP
- **Icons:** FontAwesome
- **Components:** Custom UI components with glassmorphism
- **Routing:** React Router DOM

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/kaushalsahu07/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Production Build

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Deployment

This project is ready for deployment. It's currently deployed on GitHub Pages:

```bash
npm run deploy
```

Make sure to update the `homepage` field in `package.json` with your own domain.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
- `npm run deploy` - Deploy to GitHub Pages
